# VerticalPaginator

縦書きの文章をページ分割して、ページ送りできるようにします。もともとはAozoraBookcase用に作りはじめたものですが、中核部分は特にAozoraBookcaseに依存するわけではないので別途公開します。

発想は単純で、まず縦書き１ページでブラウザにレイアウトさせたあと、1ページの幅に合わせたRangeを作っていき、各Rangeから各ページのHTMLを構築するものです。レイアウトは一回だけなので、自分でscrollWidthなどを図りながらページ分割点を見つけるより遥かに高速です。Rangeを見つければ、各ページのスクロール幅もわかるので、それを利用して１ページずつスクロールすればいいのではとも思いますが、誤差などのせいか、ページ先頭行の右半分が見えなくなったりします。おそらく対処可能と思いますが、このスクリプトではページ単位のHTMLを新しく構築します。

## 使い方
```vpcore.js```がページ分割とHTMLの構築をします。

```vpUI.js```はページ送りなどを可能にするための設定。

この２つで一応動きます。```skeleton.html```の```<template>```のなかに変換したいHTMLをペーストすれば準備完了です。あとはサーバーで配信すればOKです。例を２つ```docs/```フォルダに入れてあります。[https://adamay909.github.io/verticalpaginator/](https://adamay909.github.io/verticalpaginator/)　から表示できます。

```paginator.css```は必須です。

```vpPresistence.js```は変換したものをindexDBに保存したり、読書位置を記憶する機能を提供します。

```verticalPaginator.js```は上記のスクリプトを一箇所から制御できるようにするもの。

```vpAozora.js```はAozoraBookcaseとの連携用ですが独立しても使えるはずです。上に無い機能として、目次の追加とフォントサイズと行間の変更ができます。

かなり行きあたりばったりに作成したので、必要以上に複雑だったり、エレガントとはとても言えない状態です。悪しからず。
